## Problem Description
Convert the hexadecimal value 0x70 into its ASCII equivalent.

---

## Approach
The challenge tests understanding of how hexadecimal values map to ASCII characters. By converting the hexadecimal value to decimal, the corresponding ASCII character can be identified.

---

## Solution Steps

1. Convert 0x70 from hexadecimal to decimal: `(7 × 16^1) + (0×16^0) = 112`
2. ASCII value _112_ corresponds to the character '_p_'
3. Submit the flag in the proper flag format: **picoCTF{<ASCII_Char>}**
