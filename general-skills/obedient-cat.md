## Challenge Information
- **Name:** Obedient Cat
- **Author:** Lt 'Syreal' Jones
- **Category:** General Skills
- **Difficulty:** Easy

---

## Problem Description
The provided file contains the flag in plain sight, without any obfuscation or protection.

---

## Approach
Since the flag is stored as readable text, the only requirement is to view the contents of the file using a basic file-reading command.

---

## Solution Steps
1. Download the file using `wget`.
2. Display the contents of the file using `cat` to reveal the flag.
