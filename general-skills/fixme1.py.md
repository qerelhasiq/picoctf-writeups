## Problem Description
Fix the syntax error in this Python script to print the flag.

---

## Approach
Identify the syntax error by running the script and analyzing the Python traceback. Python errors often include the line number and type of error, which helps locate the problem quickly.

---

## Solution Steps
1. Download the Python script: `wget <link-address>`
2. Run the script to see the error: `python <python-script>.py`
3. Observe the error message: `IndentationError: unexpected indent`
4. Note the line number where the error occurs: `File "path/directory", line 20`
5. Edit the script to fix the error: `nano <python-script>.py`
6. Remove the unexpected indentation at the specified line and save the file.
7. Run the script again: `python <python-script>.py`
8. The flag is printed.
