## Challenge Information
- **Name:** FANTASY CTF
- **Author:** Lt 'Syreal' Jones
- **Category:** General Skills
- **Difficulty:** Easy

---

## Problem Description
This challenge is a short, interactive terminal game designed to introduce players to PicoCTF rules and basic CTF etiquette. You navigate the game by choosing options and following guidance from characters like Nyx and Eibhilin. The goal is to complete the game and locate the hidden flag while following the competition rules.

---

## Approach
The challenge is mostly about **following the instructions carefully** and understanding CTF best practices. Key points include:
- Registering a single private account  
- Not sharing accounts or flags  
- Completing the “sanity” challenge, which is an interactive fiction game  
- Observing clues in the dialogue to find the flag  

Since this is a guided, low-risk “sanity” challenge, the approach is to **choose the correct options** and read carefully for the flag.

---

## Solution Steps
1. When prompted about accounts, choose: c to register a single private account.  
2. When prompted for the next step, choose to play the game: a instead of trying to search externally.  
3. Follow the in-game dialogue, which guides you through the interactive fiction.  
4. At the end of the simulation, read the instructions carefully. The flag is provided directly in the game
5. Submit the flag for points
