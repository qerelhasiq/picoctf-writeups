# picoctf-writeups
Personal write-ups for PicoCTF challenges documenting my learning process in cybersecurity and CTF problem solving.
The goal of this repository is to document my problem-solving approach, tools used, and key lessons learned while developing practical cybersecurity skills.
These write-ups are intended for learning and portfolio purposes, and all challenges were solved ethically in a controlled environment.

## Repository Structure
picoctf-writeups/<br>
├── general-skills/<br>
├── web-exploitation/<br>
├── forensics/<br>
├── reverse-engineering/<br>
├── cryptography/<br>
└── binary-exploitation/

Each folder contains individual Markdown files with the following structure:
- Challenge Name
- Author
- Category
- Difficulty
- Problem Description
- Approach and Solution Steps

## Why PicoCTF?
PicoCTF provides hands-on exposure to real-world security concepts in a safe and structured environment.  
Through these challenges, I aim to strengthen my analytical thinking, attention to detail, and familiarity with common security tools and techniques.

## Disclaimer
These write-ups are for **educational purposes only**.  
No vulnerabilities were exploited outside of legal, authorized CTF environments.
